# 2. Backend Development (Express.js + SQLite)

## 2.1 Framework
Backend is built using:
- Node.js
- Express.js
- SQLite (better-sqlite3)
- Joi validators

## 2.2 Features Implemented
- GET all checks
- POST create new check
- PUT update check fields
- DELETE specific check
- Error handling middleware
- Validation middleware
- Modular routing structure

## 2.3 Folder Structure
backend/
│── routes/checks.js  
│── db.js
